Color-scheme: 
We are using the color black for the background. 
For the header we used the white color.
For the text we used white as a color. 
For the background animation we used a neon green color. 
We have some gray for the images. 

Typography: 
For the logo name we used Anonymous Pro as a font and monospace. 
The rest of the text is written in the family font Times New Romans.
The font size for the name of our compagny is 70px. 
For the about page, the font size of the text is 40px. 
For the contact page the font size is 16px. 
For the feedback page the font size is 12px.

The website consist of a camera that are fully centered
For the about page, the image is on the left and text is on the right 
For all the pages we used a border radius
